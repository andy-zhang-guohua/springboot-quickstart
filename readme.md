# 学习/演示基于 MyBatis 的一些知识点

* 项目特点 :
    * 单数据源
    * MyBatis
    * MySQL


* 学习/演示知识点 :
    * Entity 实体定义 : 完全 POJO, 不包含领域模型属性<-->表字段
    * DAO Mapper 定义 ： 完全 接口， 指定领域模型属性<-->表字段映射,指定 原生 SQL 语句
    * 组件识别 : 应用上注解 @MapperScan
